# Data & Licensing Guidelines

- Code is MIT-licensed (see `LICENSE`).
- Do **not** bundle or copy Stardew Valley Wiki text. Link to the wiki instead.
- When adding external datasets:
  - Store them under `data_external/<source>/`.
  - Add `ATTRIBUTION.md` in that folder with source, license, and retrieval date.
  - Note the dataset and license briefly in `README.md`.
- Third-party APIs keep their own licenses/ToS; review before use.
- Stardew Helper is community-made; not affiliated with or endorsed by ConcernedApe.
