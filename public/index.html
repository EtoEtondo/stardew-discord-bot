Stardew Helper

A cozy Discord bot for Stardew Valley fans. Quick wiki links, friendly slash commands, and zero clutter.

Invite to your server View on GitHub Setup guide

What it does

  • Instant `/wiki` links to the official Stardew Valley wiki (English & German).
  • Shortcuts for crops, fish, NPCs, and handy planning tools.
  • Optional update pings so your server knows when the bot changed.
  • Privacy-friendly: no message content intent; uses slash commands only.
